CSS Snippet Cheatsheet

This is an conveinent way to grab CSS syntax so you don't have to remeber. Select any snippet and it will automatically copy the code.

Flexbox Row

Use these three properties to create a Flexbox row layout.

 
    .row {
        display: flex; 
        flex-direction: row; 
        flex-wrap: wrap; 
    }
Flexbox Column

Use this to create a Flexbox column layout.

    .column {
        display: flex
        flex-direction: column;
    }
CSS Grid Layout

Build a 12-column layout using CSS Grid.

    .grid {
        display: grid;
        width: 100%;
        grid-template-columns:
        repeat(12, 1 fr);
    }
Linear Gradients

This will create a background linear gradietn from top to bottom.



    .code-container { 
        background-image: linear-gradient
        (rgba(10, 10, 10, 0.3) 0%,
        rgba(226, 22, 158, 0.644) 100%);}
Box Transition Glow

Use transition and box shadows to glow on hover.

    .code-card h2 {
        border-radius: 8px;
        transition: all 0.5s ease-in-
        out;
    }
    
    .card:hover,
    .card:hover h2 {
        box-shadow: inset 0px 0px 8px
        rgba(232, 102, 236, 1), 0 0 15px
        rgba(232, 102, 236, 1);
    }
Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point.



    .card h2 {
        position: relative;
        margin-top: -20px;
    }
